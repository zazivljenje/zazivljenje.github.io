Pričevanja ljudi, ki so izbrali življenje

Pričevanja - za življenje! je prostor za osebne zgodbe tistih, ki so se soočili s težkimi odločitvami in izbrali življenje. Tu najdete pričevanja mater, ki so kljub pritiskom obdržale otroke, staršev, ki so izgubili otroke in našli pot do ozdravljenja, ter posameznikov, ki so preživeli splav ali bili priča čudežem življenja.

Te zgodbe niso le osebne izpovedi – so pričevanja upanja, ki lahko spremenijo srca in rešijo življenja. Vsaka zgodba je dragocena, vsako življenje šteje.
